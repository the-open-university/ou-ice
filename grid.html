The reason we use "ou-grid__4" is that each row is divided into 12 columns, so if you want a three column layout with all equal widths, you divide 12 by 3 which gives you 4.

Tablet two column layout

Whatever type of layout you have, if you want to make it so every row has two columns for tablet screen sizes (between 401px and 767px), add the class ou-grid--tablet to the container.
