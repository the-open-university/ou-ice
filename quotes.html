Lighter backgrounds

For quotes with light background colours we need an extra class on the container so we can make the text darker in those scenarios. The extra class is called "ou-quote--light".

All colour options

Here is a list of the current background colours available for the quotes. You simply add this class to the blockquote container. For lighter backgrounds, two classes are required (see above).

/* Darker background, white text */
.ou-bg-green
.ou-bg-red
.ou-bg-dark-purple
.ou-bg-blue
.ou-bg-dark-blue
.ou-bg-dark-grey
.ou-bg-purple
.ou-bg-orange

/* Lighter background, black text. Requires extra ou-quote--light class */
.ou-bg-yellow
.ou-bg-light-blue
.ou-bg-grey
.ou-bg-light-grey
