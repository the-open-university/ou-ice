Lighter backgrounds

For quotes with light background colours we need an extra class on the container so we can make the text darker in those scenarios. The extra class is called ou-quote--light.

Also, as a small variation, this first example has a link in the citation instead of the read more box. This looks better for the white on white because there is no visible border for the read more box to sit flush against.

Recommended colour options

Here is a list of recommended background colours for the quotes. Simply add this class to the blockquote container. For lighter backgrounds, two classes are required (see above).

/* Darker background, white text */
.ou-bg-green
.ou-bg-red
.ou-bg-blue
.ou-bg-dark-blue
.ou-bg-dark-grey
.ou-bg-purple
.ou-bg-orange
.ou-bg-pink
.ou-bg-burgundy

/* Lighter background, black text. Requires extra class: ou-quote--light */
.ou-bg-white
.ou-bg-light-blue
.ou-bg-light-grey

With pictures

Add images to the quote using the standard class ou-pull-right to float the picture right. Recommended image size: 120px x 120px (square).
