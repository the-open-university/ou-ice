Creative

A creative allows a 100% width "showpiece" in the header area where you can place a large image, video, hero, carousel etc.

The creative will only take up the full width of the page if placed in the header (#ou-site-header).

You can have an optional text overlay, seen above, or leave it out and just feature a decorative image.

Image only

With message (seen above)

If you include a text overlay message it will align middle left by default. Add either ou-creative__message--top or ou-creative__message--bottom to move the message up or down.

Colours

The default background colour for the creative space is a light blue/grey but can be changed using a background colour class on the container (<div class="ou-creative ou-bg-red">). You will only see any of this colour if the content you place in the creative is smaller than the full width of the browser window.
