Creative

A creative allows a 100% width "showpiece" in the header area where you can place a large image, video, hero, carousel etc.

The creative will only take up the full width of the page if placed in the header (#ou-site-header).

Colours

The default background colour for the creative space is a light blue/grey but can be changed using the standard background colour classes on the container (div class="ou-creative ou-bg-green"). You will only see any of this colour if the content you place in the creative is smaller than the full width of the browser window.
