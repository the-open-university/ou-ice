Icon classes

Add these classes to your links to bring in the relevant icon:

.ou-pdf 	// PDF
.ou-doc 	// Word
.ou-xls 	// Spreadsheet
.ou-ppt 	// PowerPoint
.ou-download 	// Standard download link
.ou-rss 	// RSS
.ou-print 	// Print
.ou-info 	// Information
.ou-help 	// Help
.ou-warning 	// Warning

Inline links

Perhaps you want to link to a PDF document as well as a Word Template followed by an Excel Spreadsheet. Then don't forget the PowerPoint Presentation!

Document lists

If all items in the list require the same icon, you use two classes on the containing <ul>. To mix and match document types, use one class on the containing <ul> and specific document type classes on the <a> elements.
