Here you will find the different classes available to add icons to your pages.

Icon classes

To add an icon you need to use two classes: ou-icon + specific icon class (see below).

ou-icon ou-icon--pdf 		// PDF
ou-icon ou-icon--doc 		// Word
ou-icon ou-icon--xls 		// Spreadsheet
ou-icon ou-icon--ppt 		// PowerPoint
ou-icon ou-icon--download 	// Standard download link
ou-icon ou-icon--rss 		// RSS
ou-icon ou-icon--print		// Print
ou-icon ou-icon--info 		// Information
ou-icon ou-icon--help 		// Help
ou-icon ou-icon--warning 	// Warning

Icon without link

To use an icon in a sentence without a link you can use a span.

Document lists

If all items in the list require the same icon, you use two classes on the containing <ul>. To mix and match document types, use one class on the containing <ul> and specific document type classes on the <a> elements.
