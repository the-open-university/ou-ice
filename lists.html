If you want to use multiple links without duplicating the same icon over and over, you can simply enter paragraph tags:

  • Follow:

    @OpenUniversity

    @OU_Business_School

    @OU_Trending

The list items don't have to be links. Or they could have inline links like this:

  • Follow Arts on Facebook
  • Follow @OUArtsFaculty on Twitter

You can also combine with the ou-list--horizontal class for a linear display:

  • Watch videos
  • Latest pics
