Font Awesome is used to pull in the icons, using the icon stack feature. You can use any icon from this page.

Social network icons

  • Facebook
  • Twitter
  • LinkedIn

Add the following classes to the list item (<li>) in order to get the correct social media icon colour:

  • Facebook = ou-icon-links-facebook
  • Twitter = ou-icon-links-twitter
  • LinkedIn = ou-icon-links-linkedin

Other icons

For different icons, find the icon name on this page and put it in the list item as shown below in bold:

<ul class="ou-icon-links">
<li><span class="fa-stack">
<i class="fa fa-circle fa-stack-2x"></i>
<i class="fa fa-rocket fa-stack-1x fa-inverse"></i>
</span> <a href="#">This is a rocket ship</a></li>
</ul>

  • This is a rocket ship

Colour it

Make the background circle of your icons a different colour by specifying a colour class from the OU pallette:

<ul class="ou-icon-links">
<li><span class="fa-stack">
<i class="fa fa-circle fa-stack-2x colour1"></i>
<i class="fa fa-rocket fa-stack-1x fa-inverse"></i>
</span> <a href="#">This is a green rocket ship</a></li>
</ul>

  • This is a green rocket ship

To use a colour not in the OU pallette you can insert an inline style like so:

<ul class="ou-icon-links">
<li><span class="fa-stack">
<i class="fa fa-circle fa-stack-2x" style="color:#F00"></i>
<i class="fa fa-rocket fa-stack-1x fa-inverse"></i>
</span> <a href="#">This is a bright red rocket ship</a></li>
</ul>

  • This is a bright red rocket ship
