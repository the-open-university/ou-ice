Hero

A hero is a big image with some overlaying text, used as a promotional or feature block, or a carousel. The default colour for the text section is red but you can change this using a background colour class (see below).

Recommendations: Crop your images at 1180px wide and between 310-360px high. If using a description keep it to a maximum of two lines of text (or overlapping may occur at smaller screen sizes.)
