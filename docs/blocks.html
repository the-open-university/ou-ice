Blocks are a way to float same-sized blocks of content within the same container with no restrictions on how many you have.

You could do a similar thing using the grid but that requires separate divs for each row of content, and that's not very useful if, for example, you have a long list of items you want to keep together in the same container. The other advantage of Blocks is that you don't have to specify the size of each block - you simply add a class to the container to say 2, 3, 4 or 5 columns and then the blocks within automatically adjust their size accordingly.

Note: All blocks layouts become two columns for tablet/mobile screen sizes (unless you specify a one column layout - see final examples).

As list items

In the above examples all the blocks are divs. You can also achieve the same thing using an unordered list. The same column counts are available - 2, 3, 4 and 5 - just add the correct class to the container as with the div examples.
