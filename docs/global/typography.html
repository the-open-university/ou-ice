Font stack

The font stack for standard text and headings is:

Arial,Helvetica,sans-serif;

Inline text elements

The a tag can be used to create an inline link.

The mark tag can be used to highlight text.

Underline me

You can use the small tag for fine print.

Render this bold

Emphasise text by using the em tag.

Italicize text with no semantic emphasis by using the i tag.

Depreciated warning

It is no longer recommended to use the inline del tag as it is not announced by most screen reading technology in its default configuration. If there is a requirement to have a visual line through text then the following techniques could be explored.

04/07/2019 - V5 updates

Text classes

Introduction

The ou-introduction class can be used to increase the text size of the first paragraph for a block of text. This helps draw the eye to the beginning of the content.

Smaller headings

In some instances, it's useful to have smaller headings, for example, if you have a h2 in the sidebar which should have less importance than a h2 in the main content.

You can make headings 1-4 a bit smaller with the ou-smaller class.
