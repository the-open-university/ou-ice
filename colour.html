Note: you should always make sure the colour combinations you are creating provide enough contrast for any text to be fully legible. There is an online tool for checking the accessibility of foreground/background combos.

Colour

Change the CSS "color" property on an item to use an OU pallette colour:

  • .colour1 = green
  • .colour2 = red
  • .colour3 = gold
  • .colour4 = dark purple
  • .colour5 = mid blue
  • .colour6 = light blue
  • .colour7 = dark blue
  • .colour8 = dark grey
  • .colour9 = purple
  • .colour10 = mid grey blue
  • .colour11 = light grey blue
  • .colour12 = orange

Background colour

  • .bg-colour1 = green
  • .bg-colour2 = red

  • .bg-colour4 = dark purple
  • .bg-colour5 = mid blue

  • .bg-colour8 = dark grey

  • .bg-colour10 = mid grey blue

  • .bg-colour12 = orange
