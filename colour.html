Colour classes

Note: you should always make sure the colour combinations you are creating provide enough contrast for any text to be fully legible. There is an online tool for checking the accessibility of foreground/background combos.

Text colour

Change the CSS "color" property of an item to use an OU pallette colour:

.ou-text-green
.ou-text-red
.ou-text-dark-purple
.ou-text-blue
.ou-text-dark-blue
.ou-text-dark-grey
.ou-text-purple
.ou-text-orange
.ou-text-yellow
.ou-text-light-blue
.ou-text-grey
.ou-text-light-grey
