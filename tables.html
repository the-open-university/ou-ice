Horizontal scroll for smaller screens

Wrap the table in a div with class ou-table-wrap for tables with lots of columns. This makes sure all the data is available for smaller screen sizes by adding in a horizontal scroll.

Specify column widths

2-5 columns tables

To specify equal width columns add one of the following classes to the <table>

.ou-table--2cols /* makes all cols 50% wide (for 2 col tables) */
.ou-table--3cols /* 33% wide (3 col tables) */
.ou-table--4cols /* 25% wide (4 col tables) */
.ou-table--5cols /* 20% wide (5 col tables) */

Individual columns

To specify widths for individual columns use these classes on the <th>


w-10 /* makes the column 10% wide */
